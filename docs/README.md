# Documentation for LLMs

This directory contains **public documentation** intended for LLM consumption. All files in this directory are gitignored and should contain information that helps LLMs understand and work with Agent Patterns.

## Contents

### Standards & Compliance
- [`STANDARDS_REFERENCE.md`](./STANDARDS_REFERENCE.md) - Complete reference for rams.ai, ui-skills.com, and Vercel Design Guidelines
- [`COMPLIANCE_AUDIT.md`](./COMPLIANCE_AUDIT.md) - Full compliance audit of all patterns
- [`COMPLIANCE_CHECKLIST.md`](./COMPLIANCE_CHECKLIST.md) - Reusable compliance verification checklist
- [`STANDARDS_INTEGRATION.md`](./STANDARDS_INTEGRATION.md) - Developer guide for integrating with standards

## Purpose

These documents help LLMs:
- Understand the standards Agent Patterns follows
- Verify compliance when making changes
- Integrate patterns correctly with CopilotKit
- Maintain accessibility and design system compliance
